ConfigurationSummary

The component shows a summary of the configured product. It shows a list of cylinders, the selected colors and heights and the order status

Data

Name Type Default value Description
columns Array.<Object> - The table column properties

Computed

Name Type Description
filledHeights Array.<number> All cylinder-heights, that are part of the current configuration
items Set.<Object> The cylinders grouped by height and color, that are displayed in the table
settings Object The cylinder property settings including available heights and colors
leftCylinders Array.<Object> The cylinders objects on the left (A) stack
rightCylinders Array.<Object> The cylinders objects on the right (B) stack
allCylinders Array.<Object> The union of leftCylinders and rightCylinders

Events

Name Payload Type Description
onSuccessfulOrder Emit notification, that an order was successfully transmitted
