Order

This component represents a single order. All information about the order such as id, creation date and status will be displayed within a card.

Data

Name Type Default value Description
OrderStatus Object - Enum containing every possible order status

Computed

Name Type Description
selectedOrder string ID of the currently selected order
orderId string Formatted ID of this order
orderName string Name of this order
createdDate Date The creation date of this order
updateDate Date The date with the last update of this order
statusDescription string Description of the current production-status
numberOfParts number The number of cylinders used within this order
status string The current status of this order, which matches one of the values in the OrderStatus enum
isSelected string Indicating if this order is currently selected in the OrderList

Methods

# toggleSelection()

Marks the current order, if it was not selected. Otherwise unselects the current order.
