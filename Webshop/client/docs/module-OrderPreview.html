OrderPreview

The company displays a preview of a placed order. The plate with the corresponding cylinders will be displayed within a canvas.

Data

Name Type Default value Description
babylon Object - Data that is necessary for the BabylonJS render library. Includes the canvas and the scene
scene Object - Contains basic elements of the scene such as light, meshes, and materials

Computed

Name Type Description
selectedOrder Object The currently selected order that the preview will be shown for

Methods

# createScene()

Creates the product configuration scene.

  • Sets up the rendering pipeline
  • Loads the plate mesh
  • Creates all necessary cylinder meshes
  • Sets up camera and lighting
