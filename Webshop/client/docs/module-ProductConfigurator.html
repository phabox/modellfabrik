ProductConfigurator

The company displays the product configurator. It shows an empty plate whichs allows stacking cylinders of different heights on top of each other The component fetches the settings of the scene from the backend server. This includes the possible heights and colors of the cylinders and also the maximum height of each cylinder stack

Data

Name Type Default value Description
logik Object - Contains the logic of the scene. This includes the internal state of the configurator and the placed cylinders
babylon Object - Data that is necessary for the BabylonJS render library. Includes the canvas and the scene
scene Object - Contains basic elements of the scene such as light, meshes, and materials
gui Object - GUI information, such as panels and buttons

Computed

Name Type Description
settings Object The cylinder property settings including available heights and colors
leftCylinders Array.<Object> The cylinders objects on the left (A) stack
rightCylinders Array.<Object> The cylinders objects on the right (B) stack
allCylinders Array.<Object> The union of leftCylinders and rightCylinders
